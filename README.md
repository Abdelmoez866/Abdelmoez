 C++ E-Commerce System

A console-based E-Commerce simulation built with C++.
This project demonstrates basic programming concepts such as file handling, data structures, and user interaction in a terminal application.

---

  Features

-  User Login & Registration
-  Customer account system
-  Browse available products
-  Add products to shopping cart
-  Data stored using text files
-  Simple and clean console interface

---

 Technologies Used

- C++
- STL (Vector, String, etc.)
- File Handling
- Console Application

---

 Project Structure

cpp-ecommerce-system
│
├── main.cpp
├── users.txt
├── Cart.txt
├── products.txt
├── E-Commerce.exe
└── README.md

main.cpp → main source code

users.txt → stores user accounts

Cart.txt → stores cart data

products.txt → stores product information

E-Commerce.exe → compiled application
